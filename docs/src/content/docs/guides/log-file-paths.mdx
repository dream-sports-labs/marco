When log persistence is enabled, the logs will be saved to a file. The file paths differ between **Android** and **iOS**.

#### Android

The logs are saved at:

`/sdcard/Documents/PerformanceTracker/log.json`

#### iOS

The logs are saved at:

`Documents/PerformanceTracker/log.json`
