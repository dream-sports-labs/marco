After generating the performance reports, use the visualization dashboard to analyze and interpret the data.

### Starting the Visualization Server:

```bash
yarn perf-tracker visualize --port 8080
```

Once the server is running, open your browser at http://localhost:8080 (or the specified port) to view the performance data in the dashboard.

### Features of the Dashboard:

- **Performance Trends**: Visualize performance data over time.
- **Raw Data:** View detailed performance logs in a tabular format.
