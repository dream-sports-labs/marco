The `marco` CLI tool helps you generate performance reports and visualize performance data for your Android and iOS apps. It offers both command-line options and a configuration file, giving you flexibility in managing and analyzing your app's performance.

---

## **Configuration File**

To set default values for the CLI tool, create a `marco.config.js` file in the root of your project:

    <Tabs syncKey="config">
      <TabItem label="Android" icon="android">
        ```js
        // marco.config.js
        module.exports = {
        platform: 'android', 
        outputPath: '<PATH_TO_STORE_REPORT>',  // Default: ./generated-perf-reports
        port: '<PORT>',                        // Default: 8080
        dataDir: '<DATA_SOURCE_PATH_TO_VISUALIZE>'
        };
        ```
      </TabItem>
      <TabItem label="iOS" icon="apple">
        ```js
        // marco.config.js
        module.exports = {
        platform: 'ios', 
        iosPackage: '<iOSPackageName>',
        outputPath: '<DATA_SOURCE_PATH>',  // Default: ./generated-perf-reports
        port: '<PORT>',                    // Default port for the visualization dashboard
        dataDir: '<DATA_SOURCE_PATH_TO_VISUALIZE>'
        };
        ```
      </TabItem>
    </Tabs>

## **CLI Commands**

The marco CLI tool provides two main commands:

- [generate](../cli/generate-report): To create performance reports.
- [visualize](../cli/visualization): To view and analyze the performance data through the dashboard.
