The `marco` CLI tool facilitates the generation of performance reports and visualization of performance data for `Android` and `iOS` applications. It provides both command-line options and a configuration file, offering flexibility in managing and analyzing application performance.

---

## **Configuration File**

To configure default values for the CLI tool, create a `marco.config.js` file in the root directory of project:

    <Tabs syncKey="config">
      <TabItem label="Android" icon="android">
        ```js
        // marco.config.js
        module.exports = {
        platform: 'android',
        outputPath: '<PATH_TO_STORE_REPORT>',  
        port: '<PORT>',                        
        dataDir: '<DATA_SOURCE_PATH_TO_VISUALIZE>'
        };
        ```
      </TabItem>
      <TabItem label="iOS" icon="apple">
        ```js
        // marco.config.js
        module.exports = {
        platform: 'ios',
        iosPackage: '<iOSPackageName>',
        outputPath: '<DATA_SOURCE_PATH>',  
        port: '<PORT>',                  
        dataDir: '<DATA_SOURCE_PATH_TO_VISUALIZE>'
        };
        ```
      </TabItem>
    </Tabs>

:::note
- By default, output path is set to `./generated-perf-reports` and file path is set to `./generated-perf-reports/log.json`.
- By default, data source path is set to `./generated-perf-reports/log.json` but if need to visualize some other report, data source path can be changed.
:::

## **CLI Commands**

The marco CLI tool provides two main commands:

- [generate](../cli/generate-report): To create performance reports.
- [visualize](../cli/visualization): To view and analyze the performance data through the dashboard.
