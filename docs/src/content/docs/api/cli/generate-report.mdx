The generate command creates performance reports for `Android` or `iOS` app.

```bash frame="none"
yarn marco generate [options]
```

### Options

- `-p, --platform <platform>`: Specifies the platform to generate reports for (android or ios).
- `--ios-package <package>`: Required if the platform is iOS. Provide the iOS package name.
- `-o, --outputPath <path>`: Defines where the generated performance reports will be saved (default: ./generated-perf-reports).

Example:

```bash
yarn marco generate --platform android --outputPath ./custom-reports
```
